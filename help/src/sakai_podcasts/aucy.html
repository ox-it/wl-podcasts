Podcasts: Overview

What it does

The Podcasts tool allows site leaders to create, store, and distribute multimedia content to site participants via an RSS feed. Content can take many different forms, including audio lectures, video presentations, and slideshow presentations.

Key concepts

Subscription: Site participants can download podcasts individually or subscribe to an RSS feed, which allows podcasts to be downloaded automatically to a podcatcher.

File Types: Several different file types are supported within a podcast, including .mp3, .mp4, and .ppt.

Adding the Podcasts tool: You can add the Podcasts tool to your site using Site Setup. For instructions, see Adding, editing, or removing tools on the menubar using Site Setup.

Things to consider

  • You can download and view podcasts on most computers with podcatcher software. You do not need to own an iPod to view podcasts.

  • For Indiana University students, faculty, and staff, podcatcher software is available free of charge and available for download via IUware Online.
